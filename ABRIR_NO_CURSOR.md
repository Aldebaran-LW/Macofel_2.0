# 🌐 Como Abrir no Navegador do Cursor

## Método 1: Usando o Comando do Cursor

1. **Pressione** `Ctrl+Shift+P` (Windows/Linux) ou `Cmd+Shift+P` (Mac)
2. **Digite**: `Simple Browser: Show` ou `Simple Browser`
3. **Cole a URL**: `http://localhost:3000/admin/hero-images`
4. **Pressione Enter**

## Método 2: Usando Atalho de Teclado

- Pressione `Ctrl+Shift+B` (pode variar dependendo da configuração)
- Ou procure por "Browser" na paleta de comandos

## Método 3: Via Terminal Integrado

1. Abra o terminal integrado do Cursor (`Ctrl+`` ou View > Terminal`)
2. Execute:
```bash
cd nextjs_space
npm run dev
```
3. Aguarde o servidor iniciar (aparecerá "Ready" no terminal)
4. No navegador do Cursor, acesse: `http://localhost:3000/admin/hero-images`

## Método 4: Abrir URL Diretamente

1. No Cursor, pressione `Ctrl+Shift+P`
2. Digite: `Open URL`
3. Cole: `http://localhost:3000/admin/hero-images`
